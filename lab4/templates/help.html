Справочная система

Назначение

Приложение предназначено для автоматического семантико-синтаксического анализа текстов на английском языке.

Быстрый старт

  1. Загрузить документ для анализа на главной странице
  2. Перейдите на вкладку история, чтобы открыть проанализированный документ
  3. В результате анализа можно:
    • Редактировать исходный текст
    • Просмотреть исходный текст
    • Просмотреть морфологический анализ
    • Просмотреть синтаксический анализ
    • Просмотреть семантический анализ

Поддерживаемые форматы

  • DOC
  • DOCX
  • TXT
  • RTF
  • PDF
  • HTML

Возможности

  • Загрузка и анализ документов
  • Просмотр токенов, лемм, частей речи и синтаксических зависимостей
  • Семантический анализ через WordNet (синсеты, определения, синонимы, примеры)
  • Редактирование исходного текста
  • Сохранение результатов в базе данных SQLite
  • Экспорт в формат JSON
