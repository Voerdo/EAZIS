Руководство пользователя

Загрузка документов

  1. Перейдите на главную страницу
  2. Выберите PDF-файл
  3. Нажмите "Загрузить и обработать"
  4. Система автоматически извлечет уникальные слова и проанализирует их

Редактирование лексем

Каждая лексема содержит:

  • Лемму - начальная форма слова
  • Часть речи - грамматическая категория слова
  • Основу - неизменяемая часть слова
  • Словоформы - набор окончаний с грамматической информацией

Вы можете редактировать все эти параметры через интерфейс просмотра лексемы.

Поиск и фильтрация

  1. Ввести лемму в поле поиска
  2. Выбрать часть речи
  3. Нажмите "Применить"

Генерация словоформ

Система позволяет генерировать словоформы двумя способами:

  1. Автоматически - при загрузке PDF система автоматически определяет все формы слова
  2. Вручную - введите окончание на странице лексемы, чтобы сгенерировать форму

Сохранение и экспорт

  • Данные хранятся в формате JSON
  • Данные можно сохранить в формате TXT
